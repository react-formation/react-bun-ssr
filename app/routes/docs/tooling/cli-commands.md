# CLI commands

Core framework commands:

- `rbssr init`
- `rbssr dev`
- `rbssr build`
- `rbssr start`
- `rbssr typecheck`
- `rbssr test`

Docs workflow commands:

- `bun run docs:dev`
- `bun run docs:check`
- `bun run docs:build`
- `bun run docs:preview`

## Example session

```bash
bun install
bun run docs:check
bun run docs:build
bun run docs:preview
```

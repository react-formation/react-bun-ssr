# Sandbox Starter

Minimal Bun-native SSR starter used by the repository one-click sandbox links.

## Run locally

```bash
bun install
bun run dev
```

Then open `http://127.0.0.1:3000`.

## Build check

```bash
bun run build
```

If a browser sandbox runtime does not boot correctly in your environment, use local Bun as fallback.
